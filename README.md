
**Employee Absence Visualization Dashboard**

## Overview

This project is a data visualization dashboard designed to explore and analyze employee absence data within a company. The dashboard focuses on visualizing absence rates and types across different cities, departments, and service lengths, providing a comprehensive view of absence patterns.

## Visualization Features

1. **Decomposition Tree**:
   - A hierarchical visualization that breaks down absence data into layers: Absence Average → City → Type → Service Length.
   - This tree helps in understanding how absence rates vary across different dimensions.

2. **Multilayer Donut Chart**:
   - A multi-dimensional visualization that displays the distribution of absence grade.
   - Each layer provides insights into contribute absence rate between male and female gender.

3. **Funnel Chart**:
   - Illustrates the progression or attrition through different types of absence management .
   - Helps in identifying bottlenecks or areas where absence management can be improved.

4. **Interactive City Filter**:
   - A button slider that allows users to filter data by city.
   - Enables easy comparison of absence patterns across different locations.

## Purpose

This project aims to practice and demonstrate advanced data visualization techniques, specifically:
- Implementing decomposition trees for hierarchical data analysis
- Creating multilayer donut charts for complex data representation
- Integrating interactive elements like button sliders to enhance user experience

## Data Overview

The dashboard utilizes company data related to:
- Absence rates
- Absence types
- Employee locations (cities)
- Service length of employees

## How to Use

1. Use the button slider to filter data by city.
2. Explore the decomposition tree to understand absence patterns across different dimensions.
3. Refer to the multilayer donut chart for a quick overview of absence type distribution.
4. Analyze the funnel chart to understand absence progression or attrition.
